# Photoshop-Snippets
Schnippsel die bei der Arbeit mit Photoshop hilfreich sind.

## Hilfslinien einblenden / ausblenden
Alle Extras:
* Menü - Ansicht - Extras
* [Strg] + [H]

Nur Hilfslinien:
* Menü - Ansicht - Anzeigen - Hilfslinien
* [Strg] + [,]

## Farben, Maße, Dateigröße anzeigen (Info Fenster)
* Menü - Fenster - Info
* [F8]
